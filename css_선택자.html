선택자의종류

html파일 css 를 통해 꾸밀수 있고 꾸미기 위해서는 지정된 선택을 해야한다.

작성법

선택자 { 속성명 : 속성값; }

선택자종류 작성방법 설명
전체선택자 * HTML 내부의 모든 태그를 불러서 설정한다.
태그선택자 태그명 HTML 내부의 불려진 모든 태그를 설정
아이디선택자 #아이디이름 속성값 ID의 이름을 불러서 특정 ID만 설정
클래스선택자 .클래스명 속성값 class의 이름을 불러서 특정 클래스만 설정
id와 class의 차이점 :
id는 한페이지에 한번만 사용, 중복사용 안됨 한번 사용한 이름은 두번사용 안됨
class 는 중복적용 가능, 한 클래스에 여러 클래스명사용가능
클래스명 한칸 띄어쓰기로 여러 클래스명 부여가능
